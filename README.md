# Introduction

Flappy Doge is a delightful game inspired by two iconic elements:

- Kabosu, the Shiba Inu behind the Doge meme, which has been a cultural phenomenon since 2013.
Flappy Bird, the famous Vietnamese mobile game developed by Nguyen Ha Dong in 2013.
- The game was developed using SDL2 and C++, with a focus on delivering a fun and nostalgic experience.

_* You may not need to install SDL2, SDL_Image, SDL_TTF and SDL_Mixer because the **.dll** files are already in the ./res/sdl path._
